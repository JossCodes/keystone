# Keystone CLI

Keystone is a programmable CLI for surgical backend scaffolding, focused on clean architecture and DX-first workflows.

## CLI Dependencies

The following packages are used to support the core CLI functionality:

- **commander** — Command-line argument parsing and command routing.
- **chalk** — Terminal string styling (e.g., color, bold).
- **ora** — Elegant loading spinners for CLI feedback.
- **fs-extra** — Simplified file system operations with promise support.
- **zod** — Schema-based validation for config files and inputs.
